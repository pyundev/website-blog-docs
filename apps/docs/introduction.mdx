## Getting Started with Datamizu

Welcome to Datamizu, your serverless data workspace. Learn how to harness the power of data processing and analytics without infrastructure management.

### What is Datamizu?

Datamizu is a serverless platform that lets you:
- Create ETL pipelines using SQL, Python, or R
- Explore data through interactive notebooks
- Build and share data applications
- Collaborate with your team in real-time
